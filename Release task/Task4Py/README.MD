1. Написать одно регулярное выражение, которое в данной строке находит все телефонные номера вида: либо 8(***)*******, либо +7(***)*******, где * - некая цифра. (1 балл)

2. Написать свой простейший компилятор с помощью регулярных выражений и словарей. 
Нужно реализовать:
- поддержку переменных и присваивания им значений (1 балл);
- арифметические операции + - * / % (как над переменными, так и на числами) (1 балл);
- вывод значения на экран (1 балл);
- вывод сообщения об ошибке при попытке деления на ноль (1 балл).
Переменные могут принимать только целочисленные значение.
Все операции выполняются как операции над целыми числами.
Пример программы:
n = -7;
m = 8 + n;
print(n);
print(m);
k = 8/0
Вывод программы:
-7
1
ERROR: division by zero

3. Написать клиент-серверное приложение.
Клиент отправлят на сервер текстовые сообщения, не превосходящие 1000 знаков. Сервер в ответ отправляет клиенту эти же сообщения, но с буквами в нижнем регистре. При этом сервер должен поддерживать более одного подключения за раз. Для этого нежно самостоятельно изучить модуль threading (https://www.tutorialspoint.com/socket-programming-with-multi-threading-in-python) (4 балла).
