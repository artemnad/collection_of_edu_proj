1. Написать программу, которая созадёт картинку с разрешением jpg. На картинке изобразить 3 прямоугольника различных цветов. (1 балл)

2. Написать программу, которая производит инверсию цветов картинки, имя которой передаётся в качетве аргументов командной строки. (2 балла)

3. Написать прогрумму с графическим интерфейсом с использованием pyqt. В окне программы находится два текстовых поля одинакового размера, примём левое поле доступно для редактирования, а правое недоступно. При нажатии на кнопку происходит копирование текста из левого поля в правое.
Также программа содержит меню (в menubar), при нажатии на которое текст из правого поля копируется в левое. (3 балла)

4. Написать программу, которая принимает в качетве аргументов командной строки имена двух файлов и копирует содержимое первого файла во второй. Если файл, из которого осуществляется копирование, не существует, вывести сообщение об жтом. (1 балл)

5. Написать программу, которая принимает в качетве аргументов командной строки адрес (ip-адрес или адрес вида ya.ru) и номер порта. Затем программа осуществляет подключение к данному адресу по указанному порту и переходит в режим приёма/отправки сообщений, аналогичный работе netcat (версию для Windows можно скачать здесь https://eternallybored.org/misc/netcat/). (3 балла)
